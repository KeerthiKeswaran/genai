Here's an example Python function that implements a thread-safe PostgreSQL database connection pool using a connection array:

```python
import psycopg2
from threading import Lock

def create_pool():
    """
    Creates a connection pool for the specified number of connections.
    :return: A pool containing the specified number of connections.
    """
    conn_max = 100  # maximum number of allowed connections
    conn_min = 50   # minimum number of connections required to create
    pool = []        # empty list to store created pools
    lock = Lock()     # synchronized access to the connection array
    for I in range(conn_max):
        try:
            pool.append({'connection': psycopg2.connect(host="localhost", dbname="exampledb"), 'lock': lock})
        except ConnectionError as e:
            if e.args[0] != "Connection refused":  # catch error with a specific message
                raise Exception("Invalid host/database connection string!")
    return {'pool': pool}
```
